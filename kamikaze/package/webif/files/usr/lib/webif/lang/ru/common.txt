Если же значения слишком малы, то соединения могут разорваться просто от того, что не используются.

Helptext Remote Syslog => IP адрес и порт удаленного сервера журналов. Оставьте адрес пустым, если удаленный журнал не нужен. Порт установлен как $DEFAULT_log_port по умолчанию.
Syslog Marks => Отметки в журнале
Helptext Syslog Marks => Периодические отметки в журнале. Параметр определяет время в минутах между отметками. Величина 0 значит без отметок. Значение по умолчанию: $DEFAULT_log_mark.
Helptext Log Type => Будет ли журнал сохраняться в круговом буфере в памяти или в файле. Знайте, что файлы, сохраненные в файловой системе в памяти, будут потеряны, если Вы перезагрузите маршрутизатор. Значение по умолчанию: $DEFAULT_log_type.
Helptext Log File => Путь и имя файла журнала. Может быть установлено в любой файловой системе с записью. ПРЕДУПРЕЖДЕНИЕ: НЕ ИСПОЛЬЗУЙТЕ файловую систему JFFS, потому что syslog будет писать туда МНОГО информации. Можете использовать /tmp или любую файловую систему на внешнем носителе. Значение по умолчанию: $DEFAULT_log_file.
Helptext Log Size => Размер журнала в килобайтах. Будьте осторожны с размером кругового буфера, поскольку он отнимается из Вашей основной памяти. Значение по умолчанию: $DEFAULT_log_size кБ.

#logout.sh
logout_explanation => Поскольку аутентификация basic http кэшируется в веб-броузере, то не возможно автоматически завершить сеанс пользователя. Вы должны закрыть броузер, или если броузер - Firefox, то Удалить Частную Информацию, чтобы заставить его забыть информацию, которую Вы предоставили. Мы вероятно перейдем на метод аутентификации по cookie из-за унаследованной слабости аутентификации basic http, используемой во многих прошивках.
logout_reference => В качестве ссылки смотрите http://httpd.apache.org/docs/1.3/howto/auth.html
